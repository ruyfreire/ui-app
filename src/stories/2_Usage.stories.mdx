# Instalar via Github

### Yarn

```bash
yarn add https://github.com/ruyfreire/ui-app.git
```

### NPM

```bash
npm install https://github.com/ruyfreire/ui-app.git
```

### Instalando uma versão especifica pela tag da release

```bash
yarn add https://github.com/ruyfreire/ui-app.git#v1.0.0
```

_Lista completa de tags das releases: [Tags](https://github.com/ruyfreire/ui-app/tags)_

# Adicionando fontes

Importar a fonte Lato com pelo menos os pesos `100`, `300`, `400` e `700`

Exemplo importando via tag link no html. Você pode importar de outras formas swe desejar

```html
<!-- index.html -->
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Lato:wght@100;300;400;700&display=swap"
  rel="stylesheet"
/>
```

# Biblioteca de dependências

Instalar biblioteca para usar todas as funcionalidades do UI-App

```bash
yarn add styled-components
```

# Usar na aplicação

```jsx
// App.jsx
import { ThemeProviderUiApp, Button } from 'ui-app'

export const App = () => {
  return (
    <ThemeProviderUiApp>
      <Button color="primary">Botão</Button>
    </ThemeProviderUiApp>
  )
}
```

## Sobreeescrever o tema

`overrideTheme` é uma propriedade que recebe um tema `styled-components` e permite concatenar (cópia rasa) o tema padrão com um novo tema, para adicionar novas props ou sobrepor as atuais.

Lista completa de propriedades do tema padrão estão na seção [Tema](/story/introdução-tema--page)

```jsx
// App.jsx
import { ThemeProviderUiApp, Button } from 'ui-app'
import { MyTheme } from 'my-theme-styled-components'

export const App = () => {
  return (
    <ThemeProviderUiApp overrideTheme={MyTheme}>
      <Button color="primary">Botão</Button>
    </ThemeProviderUiApp>
  )
}
```
