# Instalar via Github

### Yarn

```bash
yarn add https://github.com/ruyfreire/ui-app.git
```

### NPM

```bash
npm install https://github.com/ruyfreire/ui-app.git
```

### Instalando uma versão especifica pela tag da release

```bash
yarn add https://github.com/ruyfreire/ui-app.git#v1.0.0
```

_Lista completa de tags das releases: [Tags](https://github.com/ruyfreire/ui-app/tags)_

# Adicionando fontes

Fontes necessárias:
- R-Flex
- Inter
 `Fonte nativa, caso não esteja disponivel, deverá ser importada com os mesmos pesos que demais fontes`

Importar a fonte R-Flex com ao menos os pesos `400/regular` e `600/bold`

Exemplo importando via arquivo local. Você pode importar de outras formas que desejar

```html
<!-- app.css -->
@font-face {
  font-family: "R-Flex";
  src: local("R-Flex"), url("./fonts/R-Flex/R-Flex-Regular.otf") format("opentype");
}

@font-face {
  font-family: "R-Flex";
  src: local("R-Flex"), url("./fonts/R-Flex/R-Flex-Bold.otf") format("opentype");
  font-weight: bold;
}
```

# Biblioteca de dependências

Instalar biblioteca para usar as funcionalidades da UI-App

```bash
yarn add styled-components
```

# Usar na aplicação

```jsx
// App.jsx
import { ThemeProvider } from 'styled-components'
import { theme, GlobalStyle, Button } from 'ui-app'

export const App = () => {
  return (
    <ThemeProvider theme={theme}>
      <GlobalStyle />
      {/* Seu código aqui */}
    </ThemeProvider>
  )
}
```
