## Estilo Global

Estilo global aplicado a cima do tema para normalizar o css da página

```css
* {
  box-sizing: border-box;
  margin: 0;
  outline: 0;
  padding: 0;
}

html {
  /**
  * Transforma a fonte base da página para melhorar a escrita em 'rem',
  * fazendo 1.6rem = 16px
  */
  font-size: 10px;
}

body {
  font-family: ${theme.font.family.base};
  font-size: ${theme.font.size.md};
}
```
