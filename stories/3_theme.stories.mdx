| theme.font.size.md   | 1.6rem (16px)  |
| theme.font.size.lg   | 2rem   (20px)  |
| theme.font.size.xl   | 2.4rem (24px)  |

## Estilo Global

Estilo global para normalizar o css da página e adptar o font size para o tamanho do rem

```css
  * {
    box-sizing: border-box;
    margin: 0;
    outline: 0;
    padding: 0;
  }

  html {
    font-size: 10px;
  }

  body {
    font-family: ${theme.font.family.base};
    font-size: ${theme.font.size.md};
    color: ${theme.colors.white2};
  }
```
